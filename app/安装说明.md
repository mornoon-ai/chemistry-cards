# iPhone 安装说明

这是一个适合 iPhone 使用的高中化学离线卡片 Web App（PWA）。

## 文件位置

应用入口：

```text
学习卡应用/index.html
```

图片依赖：

```text
图片/
```

因此部署时需要保留 `学习卡应用` 和 `图片` 两个目录的相对位置。

## 推荐安装方式

1. 将整个高中化学文件夹发布到支持 HTTPS 的静态网站。
2. 在 iPhone Safari 中打开：

```text
https://你的地址/学习卡应用/index.html
```

3. 点击 Safari 分享按钮。
4. 选择“添加到主屏幕”。
5. 主屏幕会出现“化学卡片”应用图标。

## 本地预览方式

在 Mac 上进入项目根目录后运行：

```bash
python3 -m http.server 8765
```

然后在浏览器打开：

```text
http://127.0.0.1:8765/学习卡应用/index.html
```

如果要在 iPhone 上通过局域网访问，把 `127.0.0.1` 换成 Mac 的局域网 IP。

## 注意

- A/B/C 状态、连续 A 次数、下次唤醒时间保存在当前浏览器本地。
- 连续三次选择 A，会退出唤醒队列。
- 若要完整离线缓存和主屏幕 App 体验，建议使用 HTTPS 部署。
- 原生 iOS App/IPA 需要 Xcode、签名证书和 Apple 开发者账号；这个版本是可作为原型继续迁移的 PWA。
